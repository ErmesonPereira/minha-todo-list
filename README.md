# 📝 Minha Lista de Tarefas (To-Do List)

Este é um projeto prático desenvolvido para exercitar conceitos fundamentais de **HTML5, CSS3 e JavaScript**. O objetivo foi criar uma aplicação de lista de tarefas funcional, intuitiva e com boa experiência de usuário.

## 🚀 Funcionalidades

O projeto foi construído de forma evolutiva, passando pelas seguintes etapas:

* **v0: Estrutura Inicial** - Criação da base HTML e estilização básica, permitindo adicionar e excluir tarefas.
* **v1: Interatividade** * Adição de `checkbox` para marcar tarefas como concluídas (com efeito visual de texto riscado).
    * Implementação da função de **Editar** descrição para corrigir tarefas já criadas.
* **v2: Experiência do Usuário (UX)** * Atalho de teclado: agora é possível adicionar tarefas apenas pressionando a tecla **Enter**.
    * Auto-foco: após adicionar uma tarefa, o campo de texto limpa e o cursor volta automaticamente para o input.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica.
* **CSS3**: Design responsivo e estados visuais (classes dinâmicas).
* **JavaScript (Vanilla)**: Manipulação do DOM e lógica de eventos.
